# Contributing

Thanks for your interest in contributing! This is a personal VR game project, but community contributions for new bosses and content are welcome.

## Before You Start

Check the [Wiki](../../wiki) for guides on how bosses are structured before making any changes. PRs that don't follow the existing patterns will likely be closed.

## What You Can Contribute

- New boss designs and implementations
- Bug fixes
- Balance tweaks (with reasoning)

## How to Submit

1. Fork the repository
2. Create a branch from `main`
3. Make your changes and test them in Unity
4. Open a pull request with a description of what you added and how you tested it

All PRs are reviewed by [Raegan Scheet](https://github.com/raedotzip). There's no guaranteed turnaround time — this is a personal project.

## Suggestions

Have an idea for a boss or game mechanic but don't want to submit code? Post it in the [Discussions](../../discussions) tab.

## Questions or Issues

For general questions, use [Discussions](../../discussions). For bugs, open an [Issue](../../issues).
