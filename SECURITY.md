# Security Policy

## Reporting a Concern

If you find a security issue or have a private concern about this project, please **do not open a public issue**.

Instead, submit a private report using [GitHub's private vulnerability reporting](../../security/advisories/new).

Alternatively, you can contact the maintainer directly through GitHub: [@raedotzip](https://github.com/raedotzip).

## Scope

This is a personal game project. Security concerns are mainly limited to:

- Malicious code in contributions (PRs, forks)
- Dependencies with known vulnerabilities

## Response

There are no guaranteed response times, but all reports will be reviewed.
